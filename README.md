# datapacks

A collection of Minecraft datapacks I've made.

## Building

Requires [EasyDatapacks](https://github.com/emorgan00/EasyDatapacks). Run
`./build.sh` and a directory and a zip for each `.mcf` file will be generated in
`dist/`.
